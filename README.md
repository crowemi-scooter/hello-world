# Hello World App

This is a simple Python application that prints "hello scooter".

## Setup

1. Clone the repository:
   ```bash
   git clone git@github.com:crowemi-scooter/hello-world.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hello-world
   ```
3. Run the script:
   ```bash
   python3 main.py
   ```
